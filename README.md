## 级与届

“级”指入学年份，“届”指毕业年份

如：2019级新生、2023届毕业生/校友

## 教务处与教务办

* 以下单位是校级行政单位
  * 教务处
  * 研究生院
  * 学生处
  * 人力资源处
  * 国际交流处
  * ……
* 以下单位是院级行政单位
  * 本科生教务办
  * 研究生教务办
  * 学生办
  * 人力资源与国际交流办
  * ...

{% hint style="warning" %}
院级没有“教务处”，只有“教务办”
{% endhint %}

## 法定节假日

* 法定节假日的放假安排
  * 由校党政办
  * 以通知的形式
  * 发布在学校主页上
* 开学、长假、考试、报到等与教学相关的时间安排
  * 由教务处
  * 以教学校历的形式
  * 发布在教务处网站上
* 交大的法定节假日通常与国务院公布的节假日安排一致
